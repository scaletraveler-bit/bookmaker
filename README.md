# Colosseo BookMaker

FF14 Colosseo用 ブックメーカーオッズ計算アプリ

## 公開URL

https://scaletraveler-bit.github.io/bookmaker/

## 機能

- 赤・青への賭け金追加
- オッズ自動計算
- 最大オッズ設定
- 履歴保存
- ページ更新後もデータ保持
